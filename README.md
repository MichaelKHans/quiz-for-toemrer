# 🏗️ Tømrer Quiz - Cloud Edition (v3.2.5)

Dette er en professionel quiz-platform udviklet specifikt til tømreruddannelsen. Systemet er nu opgraderet med fuld Cloud-synkronisering via Google Firebase.

## 🚀 Nye Funktioner
- **☁️ Cloud Synkronisering:** Alle rettelser lavet af undervisere i Lærer Panelet opdateres med det samme for alle elever og kolleger.
- **🧠 Intelligent Billed-søgning:** Systemet forstår danske fagudtryk (f.eks. "skimmel", "tagspær") og finder automatisk relevante tekniske billeder.
- **📸 Billed-lås:** Undervisere kan låse præcis det billede, de ønsker, til hver quiz.
- **📜 Underviser Logbog:** En oversigt over alle opdateringer og nye funktioner direkte i systemet.

## 🛠️ Sådan bruger du systemet
1. **Elever:** Åbn `index.html` for at se quiz-oversigten.
2. **Undervisere:** Klik på den grønne cirkel (🔒) i toppen for at åbne Lærer Panelet.
   - **Kode:** `tømrer123`
   - Her kan du oprette nye quizzer manuelt eller bruge den indbyggede **AI-assistent**.

## 💻 Teknisk Opsætning
Projektet bruger:
- **Frontend:** Vanilla HTML, CSS og JavaScript.
- **Database:** Firebase Realtime Database.
- **Billeder:** LoremFlickr (Unsplash) API.
