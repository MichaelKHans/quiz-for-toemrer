# Tømrer Quiz - Cloud Edition (v3.3.0)

Dette er en professionel quiz-platform udviklet specifikt til tømreruddannelsen. Systemet kører med fuld Cloud-synkronisering via Google Firebase.

## Funktioner
- **Cloud Synkronisering:** Alle rettelser lavet af undervisere i Lærer Panelet opdateres med det samme for alle elever og kolleger.
- **Dashboard Design:** Hvert quiz-kort på forsiden har nu et subtilt baggrundsbillede i hjørnet, der matcher emnet og giver en moderne visuel oplevelse.
- **Intelligent Billed-søgning:** Systemet forstår danske fagudtryk (f.eks. "skimmel", "tagspær") og finder automatisk relevante tekniske billeder.
- **Billed-lås:** Undervisere kan låse præcis det billede, de ønsker, til hver quiz i Lærer Panelet.
- **Underviser Logbog:** En oversigt over alle opdateringer og nye funktioner direkte i systemet.

## Sådan bruger du systemet
1. **Elever:** Åbn index.html for at se quiz-oversigten.
2. **Undervisere:** Klik på cirklen i hjørnet for at åbne Lærer Panelet.
   - **Kode:** tømrer123
   - Her kan du oprette nye quizzer manuelt eller bruge den indbyggede AI-assistent.

## Teknisk Opsætning
Projektet er bygget med:
- **Frontend:** Vanilla HTML, CSS og JavaScript.
- **Database:** Firebase Realtime Database.
- **Billeder:** LoremFlickr / Unsplash API integration.
